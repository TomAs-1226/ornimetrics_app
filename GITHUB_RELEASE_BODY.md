## What's New

**Features**
- AI Species Identifier - Identify birds from photos
- Field Observations - Log sightings with location, weather & notes
- My Field Detections - View all saved observations
- Dynamic Island Timer - Floating session timer overlay
- Data Export - CSV, JSON, and text report exports
- Photo Gallery - View feeder snapshots with detections
- Offline Mode - Access data without internet

**Improvements**
- Expanded Terms of Service
- AI Improvement Program transparency
- Better Firebase error handling
- Enhanced haptic feedback

**Fixes**
- Field detections now save correctly per user
- Timer display fixed
- Removed unused dependencies

## Installation

1. Download the APK
2. Open the file on your Android device
3. Enable "Install from unknown sources" if prompted
4. Install and enjoy!

**Requirements:** Android 5.0+
